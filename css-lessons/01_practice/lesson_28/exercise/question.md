## 2. 以下のスタイルを定義
   | セレクタ | プロパティ | 値 |
   | --- | --- | --- |
   | .button-hamburger| 4つ角の角丸 | `50%` |
   | .service | 4つ角の角丸 | `0.5rem` |
   | ^^ | オーバーフロー | 非表示 |

   | ^^ | トランジションのイージング | `ease-out` |
   | ^^ | トランジションのプロパティ | `clip-path` と `filter` |
   | .serviceがホバー時とフォーカス時の子孫の .service-image | クリッピング領域 | `inset(15% round 0.5rem)` |
