# 練習問題


## 1. 「index.html」に「execises」› 「assets」 › 「styles」フォルダにある「global.css」をリンク


## 2. 完成イメージをよく観察し、以下のヒントを参考にしながら、==ナビゲーションリンクのホバー時（フォーカス時も）の線のトランジション==と、フッター上の==ループテキストアニメーション==を作成（アニメーション時間などが違ってもOK）

   【ナビゲーションリンク】
   -  擬似要素でテキストカラーと同じ色の背景色を設定した高さ1pxのボックスを、リンクテキストの下に配置
   - 非ホバー時（フォーカス時も）の擬似要素は、右端を基準にX方向を0に縮小
   - ホバー時（フォーカス時も）の擬似要素は、左端を基準にX方向を等倍に戻す
   - 擬似要素の拡大縮小のスタイルをトランジションで繋ぐ


   【ループテキストアニメーション】
   -  進行度が0%の時は元の位置、100％の時は、X方向に-100%平行移動するキーフレームアニメーションを作成
   -  ループテキスト用の2つの同じテキストを横に並べる（1つのテキストは `aria-hidden`属性でスクリーンリーダーで読み上げられないように対策してある）
   - 各テキストに、作成したキーフレームアニメーションを設定する（等速で無限ループ）
   -  横スクロールが出ないように、2つテキストの親要素をオーバーフローする部分は非表示にする
