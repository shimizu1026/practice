## 2. 下記の指示に従ってスタイルを適用

   | 適用箇所 | プロパティ | 値 |
   | --- | --- | --- | 
   |  .header |  表示方法 | ブロックレベルのフレックスコンテナー |
   | ^^ | 主軸方向の配置 | 主軸に均等に配置（両端は間隔をあけない） |
   | ^^ | 交差軸方向のアイテム達の位置揃え | 交差軸の中央に揃える |
   |  .nav-list |  表示方法 | ブロックレベルのフレックスコンテナー |
   | ^^ | 交差軸方向のアイテム達の位置揃え | 交差軸の中央に揃える |
   | ^^ | 行のアイテム間の溝 | `0.5rem` |
   | ^^ | 列のアイテム間の溝 | `2rem` |
   |  .nav-list の子孫の `a` 要素 |  テキストの装飾 | なし |
   | ^^ | 文字の太さ | `500` |
   |  .nav-list の子孫の `a` 要素（ホバー時とフォーカス時） |  テキストの装飾 | 下線 |
   |  .post | 表示方法 | ブロックレベルのフレックスコンテナー |
   | ^^ | 列のアイテム間の溝 | `2rem` |
   | .post-body |  伸長係数 | `1` |
   | ^^ |  縮小係数 | `1` |
   | ^^ |  主軸方向の初期サイズ | `auto` |
   | .post-over |  主軸方向の初期サイズ | `50%` |
   | ^^ |  アイテムの順番 | `-1` |
