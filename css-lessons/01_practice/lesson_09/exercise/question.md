# 練習問題


## 1. 「execises」フォルダ内に「assets」 › 「styles」フォルダを作成

## 2.「styles」フォルダ内に「global.css」を作成し、「index.html」にリンク

## 3. 下記の指示に従ってスタイルを適用

   | 適用箇所 | プロパティ | 値 |
   | --- | --- | --- | 
   | `h1`、`h2`、`h3`、`h4`、`h5`、`h6` のいずれかに一致（詳細度は0にする） | 文字サイズ | 継承 |
   | ^^ | 文字の太さ| `400` |
   | body要素 | 書体 | Noto Sans JP なければ サンセリフ |
   | ^^ | 文字の色 | `#202020` |
   | ^^ | 行の高さ | `1.5` |
   | ^^ | 禁則処理 | 厳格 |
   | ^^ | 行を超えるテキスト | 行は許可された改行ポイントがない場合、改行不可能なポイントで改行 |
   | 子孫に`data-theme`属性が`dark`の` 選択中の input`要素を持っている `body`要素 | 背景色 | `#202020` |
   | ^^ | 文字の色 | `#f0f0f0` |
   | 選択範囲のテキスト | 背景色 | `#999a7d` |
   | `.nav-list` の子である `li`要素のマーカー | 文字色 | `#808080` |
   | `.nav-list` の子孫である `a`要素 | 文字色 | 継承 |
   | ^^ | テキストの装飾 | なし |
   | `.nav-list` の子孫である `a`要素のホバー時 | 文字色 |`#666` |
   | ^^ | テキストの装飾 | 下線 |
   | ^^ | テキストと下線のオフセット | `0.25em` |
   | `.heading`  | 文字サイズ | `1.5rem` |
   | `.heading`の先頭の文字 | 文字サイズ | `1.5em` |
