## 2. 以下を参考にメディアクエリを使って、ウインドウサイズによるスタイル、視差効果オプションへの配慮、ホバースタイルはタッチデバイスでは反応しないようにする。

   | セレクタ | メディアクエリ | プロパティ | 値 |
   | --- | --- | --- | --- |
   | 全ての要素（擬似要素も含む） | 視差効果を減らす機能が有効時 | トランジション | 無効 |
   | ^^ | ^^ | アニメーション | 無効 |
   | .button-hamburger | ビューポート幅が1024px以上の時 | 表示モード | 非表示 |
   | .nav-list | ビューポート幅が1024px以上ではない時 | 表示モード | 非表示 |
   | .services | - | ブロック軸のスタート方向の間隔 | `--space-lg` |
   | ^^ | ビューポート幅が768px以上の時 | ^^ | `--space-xl` |
   | ^^ | - | 行の溝 | `--space-md` |
   | ^^ | ビューポート幅が768px以上の時 | ^^ | `--space-lg` |
   | .service-cover  | - | アスペクト比 | 2 : 1 |
   | ^^ | ビューポート幅が768px以上の時 | ^^ | 2.5 : 1 |
   | ^^ | ビューポート幅が1024px以上の時 | ^^ | 3 : 1 |
   | .service-name | - | 文字サイズ | `--font-lg` |
   | ^^ | ビューポート幅が1024px以上の時 | ^^ | `--font-xl` |
   | `:hover` 擬似クラスを利用している要素 | ホバー可能デバイス | `:focus-visible` と同じスタイル | `:focus-visible` と同じ値 |
