# 練習問題

## 1. 下記の指示に従ってスタイルを適用（セレクタの種類は問わない）

   | 適用箇所 | プロパティ | 値 |
   | --- | --- | --- | 
   | body要素 | トラッキング | 0.05文字分 |
   | ヘッダーのサイト名 | 行揃え | 中央 |
   | ナビゲーションのリンク | 装飾 | なし |
   | ヘッダーのサイト名 | 書体 | Inter（Google Fonts）、 サンセリフ書体 |
   | 段落「私たちは架空の – プロ集団です。」 | トラッキング | 0.1文字分 |
   | 著作権表記 | 行揃え | 文章の方向が左から右であれば`right` 、右から左であれば`left` |
   | ^^ | 変形 | 全て大文字 |
