# 練習問題

## 1. 下記の指示に従ってスタイルを適用（セレクタの種類は問わない）

   | 適用箇所 | プロパティ | 値 |
   | --- | --- | --- | 
   | body要素 | 行からテキストが溢れる場合の処理 | 行は許可された改行ポイントがない場合、改行不可能なポイントで改行 |
   | ^^ | 禁則処理| 最も厳格なルールを使用して改行 |
   | 段落「私たちは架空の – プロ集団です。」 | 行からテキストが溢れた場合の改行の有無 | 自然なフレーズで改行 |
   | ^^ | テキストの折り返し |  各行の文字数を均等にする方法で折り返す |
   | 「Works」のhgroup要素 | 横書き・縦書き | 右から左に流れる縦書き（日本語の場合） |
