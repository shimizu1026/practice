Dummy Cafe

Dummy Cafeに一度でもご来店頂いたお客様は、次回から「いつもの。」でご注文頂けます。もちろんあなたのお好きな、コーヒーの苦さ、お砂糖やミルクの量もちゃんと把握しています。 あなたがDummy Cafeにいる間は、「いつもの。」というたったひとつの魔法の言葉で全てが通じる。そんな架空のサービスを提供しています。
